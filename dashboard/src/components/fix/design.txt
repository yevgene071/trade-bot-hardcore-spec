**Главная проблема: у чарта нет иерархии сигнала.** Всё одинаково важно → ничего не важно. Трейдер за секунду должен считать 3 вещи: куда идёт цена, с какой силой, и где стены. Сейчас ни одна не читается.Окей, это не детали патча — это проблема концепции. Рисую что должно быть.Вот что нужно сделать в коде. Три изменения в `drawPrice`, одно в `drawDensity`:

**1. Area fill под ценовой линией** — добавить в `drawPrice` после отрисовки spread shadow:

```ts
// После spread shadow, перед ghost lines:
const isUp = pts[n-1].mid > pts[0].mid;
const fillClr = isUp ? 'rgba(38,194,110,' : 'rgba(224,80,80,';
const areaGrad = ctx.createLinearGradient(0, 0, 0, H);
areaGrad.addColorStop(0, fillClr + '0.15)');
areaGrad.addColorStop(1, fillClr + '0.01)');

ctx.beginPath();
let started = false;
for (let i = 0; i < n; i++) {
  if (pts[i].mid <= 0) { started = false; continue; }
  const x = getX(pts[i].ts_unix_ms, latestTs, W);
  const y = toY(pts[i].mid, H, mn, mx);
  if (!started) { ctx.moveTo(x, y); started = true; }
  else ctx.lineTo(x, y);
}
ctx.lineTo(getX(pts[n-1].ts_unix_ms, latestTs, W), H);
ctx.lineTo(getX(pts.find(p=>p.mid>0)!.ts_unix_ms, latestTs, W), H);
ctx.closePath();
ctx.fillStyle = areaGrad;
ctx.fill();
```

**2. Цветная линия по momentum** — заменить текущий блок рисования price line:

```ts
// Вместо ctx.strokeStyle = lastAgg > 0.6 ? C.priceAgg : C.priceLine;
// рисуем посегментно:
for (let i = 1; i < n; i++) {
  if (pts[i].mid <= 0) continue;
  const agg = pts[i].tape_aggression;
  ctx.beginPath();
  ctx.moveTo(getX(pts[i-1].ts_unix_ms, latestTs, W), toY(pts[i-1].mid, H, mn, mx));
  ctx.lineTo(getX(pts[i].ts_unix_ms, latestTs, W), toY(pts[i].mid, H, mn, mx));
  ctx.strokeStyle = agg > 0.15 ? '#26c26e' : agg < -0.15 ? '#e05050' : '#7a8fa8';
  ctx.lineWidth = 2;
  ctx.stroke();
}
```

**3. Текущая цена — крупно в заголовке** — текст в `#080c16` хедере, 18px, цвет по направлению. Сейчас это теряется в `C.priceLine` 10px label на canvas.

**4. OB density — только в правую колонку** — в `drawDensity` добавить горизонтальный clip:
